Netflix Movies and TV Shows | Data analyst
Descripción:
Este dataset presenta programas de TV y películas en Netflix hasta 2019, obtenido de Flixable, un buscador externo. 
Según un informe de 2018, la cantidad de programas de TV en Netflix casi se ha triplicado desde 2010, 
mientras que las películas han disminuido en más de 2,000 títulos. Explorar el conjunto de datos puede revelar más ideas, 
y la integración con otras fuentes, como las calificaciones de IMDB y Rotten Tomatoes, puede proporcionar hallazgos interesantes.

Para el presente proyecto se realizo la descarga del https://www.kaggle.com/datasets/shivamb/netflix-shows

Instalación
 Configuración del Entorno de Python (Jupyter Notebook):
  Asegúrate de tener Python instalado en tu máquina.
  Instala Jupyter Notebook mediante el siguiente comando en tu terminal o consola.
  Abre Jupyter Notebook y ejecuta cuaderno para realizar tu análisis y exportar el conjunto de datos a SQL.

 Conexión de Datos con Power Query para Excel:
  Abre Excel y selecciona la pestaña "Datos" o "Data".
  En el grupo "Obtener y transformar datos" o "Get & Transform Data", selecciona "Desde Otra Fuente" y luego "Desde SQL Server".
  Ingresa los detalles de conexión, como el nombre del servidor, la autenticación, la base de datos y la consulta SQL si es necesario.

Conexión de Datos con Power BI:
  Abre Power BI Desktop.
  Selecciona "Obtener datos" y elige la opción que corresponda a tu base de datos SQL.
  Ingresa los detalles de conexión, como el nombre del servidor, la autenticación, la base de datos y la consulta SQL si es necesario.

Uso
 - En Excel o Power BI, crea informes y visualizaciones utilizando los datos importados.
 - Utiliza las herramientas disponibles para diseñar paneles interactivos y gráficos.
